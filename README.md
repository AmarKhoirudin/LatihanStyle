### Persyaratan

- Membutuhkan Node >= 8.10 and npm >= 5.6

### Cara menjalankan
1.  Dengan NPM
	- Buka file di terminal atau code editor
	-  ketik " npm install "
	-  ketik "npm start "
	-  Buka browser dan buka [Link](http://localhost:3000/ "Link")

2.  Dengan Yarn
	- Buka file di terminal atau code editor
	-  ketik " yarn install "
	-  ketik "yarn start "
	-  Buka browser dan buka [Link](http://localhost:3000/ "Link")
